   كيفية بناء التطبيق - HOW TO BUILD
═══════════════════════════════════════════════════════

⚠️ مهم جداً: يجب أن تكون في المجلد الصحيح!

المجلد الصحيح:
C:\Users\Dell\Desktop\guide-yourself-\smart-guide-app

═══════════════════════════════════════════════════════

الطريقة 1: استخدام السكريبت (الأسهل)
───────────────────────────────────────────────────────

1. افتح PowerShell في هذا المجلد
2. شغّل:
   .\build.ps1

═══════════════════════════════════════════════════════

الطريقة 2: الأوامر اليدوية
───────────────────────────────────────────────────────

1. افتح PowerShell

2. انتقل للمجلد الصحيح:
   cd C:\Users\Dell\Desktop\guide-yourself-\smart-guide-app

3. تأكد أنك في المجلد الصحيح:
   pwd
   
   يجب أن يظهر:
   C:\Users\Dell\Desktop\guide-yourself-\smart-guide-app

4. ابدأ البناء:
   eas build --platform android --profile preview

═══════════════════════════════════════════════════════

الطريقة 3: من VS Code Terminal
───────────────────────────────────────────────────────

1. في VS Code، اضغط Ctrl + `
2. تأكد أن Terminal في المجلد الصحيح
3. شغّل:
   eas build --platform android --profile preview

═══════════════════════════════════════════════════════

❌ الأخطاء الشائعة
───────────────────────────────────────────────────────

خطأ: "Missing script"
→ أنت في المجلد الخطأ!
→ الحل: cd smart-guide-app

خطأ: "Root-level expo object found"
→ أنت في المجلد الخطأ!
→ الحل: cd smart-guide-app

خطأ: "Build failed"
→ راجع السجلات على:
   https://expo.dev/accounts/sun32732/projects/guide-yourself/builds

═══════════════════════════════════════════════════════

   pwd

   C:\Users\Dell\Desktop\guide-yourself-\smart-guide-app

   C:\Users\Dell\Desktop\guide-yourself-

📱 بعد نجاح البناء
───────────────────────────────────────────────────────

1. ستحصل على رابط تحميل APK
2. حمّل الملف
3. انقله لهاتف Android
4. ثبّته
5. استمتع! 🎉
